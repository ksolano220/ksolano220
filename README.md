## Featured Project: Sentra

Sentra is a system designed to monitor and evaluate high-risk actions in automated workflows.

Instead of focusing only on outputs, Sentra focuses on how decisions are made and executed.

Core components:
- Action monitoring layer for tracking system behavior
- Rule-based evaluation of actions
- Risk scoring system to assess potential impact
- Intervention logic to flag or stop high-risk actions

This project reflects my interest in understanding and controlling decision-making in complex systems.
